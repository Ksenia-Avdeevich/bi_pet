# FinSight — Финансовая аналитика 

Pet-проект в Power BI: интерактивный дашборд для анализа финансовых транзакций, клиентов и рисков.

**Автор:** Авдеевич Ксения, 2026

---

## О проекте

Дашборд разработан по мотивам бизнес-требований условной финансовой организации, которой нужен был **Дашборд с финансовой аналитикой** — единая точка мониторинга транзакций, поведения клиентов, комиссий и эффективности по сегментам бизнеса и регионам.

Задача дашборда — помочь бизнесу:
- отслеживать динамику и рост транзакций в реальном времени;
- выявлять наиболее прибыльные клиентские сегменты и регионы;
- анализировать паттерны и тренды по типам транзакций;
- контролировать комиссии;
- понимать демографию клиентов;
- принимать более обоснованные финансовые решения.

## Стек

- **Power BI Desktop** (формат проекта `.pbip` — Power BI Project)
- Модель данных на DAX (мера с YoY-сравнением, динамическая метрика через параметр)

## Структура файлов

```
pet_bi.pbip          # файл-проект Power BI (точка входа)
pet_bi.Report/        # визуальный слой отчёта
```

Открывается в Power BI Desktop: `pet_bi.pbip -> Открыть`.

## Реализованные KPI

| KPI | Статус |
|---|---|
| Total Amount (общая сумма, с YoY) | 
| Total Transactions (всего транзакций) |
| Average Transaction Value (среднее значение транзакции) |
| Total Fees (общая комиссия) |


## Визуализации

- **Total Amount by Month** — линейный/area-график динамики выбранной метрики по месяцам, поиск сезонности
- **Total Amount by Transaction Status** — donut-диаграмма Success / Failed / Pending
- **Total Amount by Customer Segment** — горизонтальный bar-чарт по сегментам: Retail, Premium, SME, Corporate, Wealth
- **Total Amount by City/State** — горизонтальный bar-чарт по регионам (топ по сумме)
- **Transaction Type Analysis** — матрица по типам транзакций (Bill Payment, Card Payment, Deposit, Fee Charge, Interest Credit, Investment, Loan EMI, Refund, Transfer, Withdrawal) с показателями: сумма, комиссия, кол-во транзакций
- **Gender Analysis** — donut-диаграмма по полу клиентов

## Интерактивность / фильтры

- **Год** (2023–2026)
- **Динамическая метрика** — переключение отображаемого показателя (общая сумма / общая комиссия / всего транзакций / отдельные категории трат)
- **Профессия** (Occupation)
- **Категория продавца** (Category)

Все визуалы синхронно перестраиваются под выбранные срезы, KPI-карточки показывают разницу год к году (YoY, %).
